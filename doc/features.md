# Xususiyatlar — Vizitka & Web

## Tab strukturasi (yakuniy)

Tab 1: Haqida      → Kim ekanligim
Tab 2: Aloqa       → Menga qanday murojaat / tolash
Tab 3: Ijtimoiy    → Meni qayerda topish mumkin
Tab 4: Qoshimcha   → Xizmatlar, portfolio, CV (egasi yoqsa korinadi)

Eslatma: Karta raqami — Aloqa tabida, ekranda KORINMAYDI.
Faqat [Nusxalash] tugmasi korinadi. Bosganda clipboard ga boradi.
