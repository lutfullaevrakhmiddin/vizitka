# Dizayn — Vizitka & Web

## Asosiy tamoyil: Mobile-First
- 80% foydalanuvchi telefonda
- Minimum tugma olchami: 44x44px
- Bir qol bilan ishlatish imkoni
- Max kenglik: 430px (katta ekranda markazda)

---

## Sahifa tuzilmasi

Landing page:
  Hero          → sarlavha + telefon ichida vizitka demo
  Qanday ishlaydi → 3 qadam (ikonka + qisqa matn)
  Ishonch       → Xavfsiz, Bepul, 1 daqiqada tayyor
  Demo vizitka  → jonli namuna
  CTA           → Bepul yaratish tugmasi

Vizitka sahifasi:
  Yuqori        → avatar + ism + kasb
  Tab bar       → Haqida | Aloqa | Ijtimoiy | Qoshimcha
  Kontent       → tanlangan tab malumotlari
  QR kod        → pastda, kichik, bosganda katta ochiladi
